# manga-go-app

**NOTE**: super incomplete

This is a simple server for providing manga data from MangaEden.

### Getting started

Install dependencies:
```bash
go get -u github.com/golang/dep/...
dep ensure -v
```

Start up docker:
```bash
./start-docker.sh
```

To start the server:
```bash
./run-server.sh
```

### Testing

To run tests:
```bash
./run-tests.sh
```
